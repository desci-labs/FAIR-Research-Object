FAIROS-metric-I1-02

This metric ensures that metadata makes use of semantic resources—such as controlled vocabularies, ontologies, and thesauri—to enhance interoperability and machine readability to the Research Object.

What is being measured?

Whether the metadata includes references to semantic resources

Why should we measure it? Using semantic resources ensures that metadata terms have consistent meanings, enabling semantic interoperability across systems and domains.

What must be provided for the measurement?

The metadata record.

What are considered valid results?

The metadata uses terms or elements from recognized semantic resources, with dereferenceable URIs.
