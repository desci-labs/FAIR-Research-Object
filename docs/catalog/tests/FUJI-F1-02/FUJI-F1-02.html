FUJI-F1-02

This test assesses whether the data uses a valid and persistent identifier scheme, based on the IDTUILS library. Unlike test FAIROS-F1-01, this test focuses on the data itself, not the metadata. It checks whether the data is included as one of the metadata elements.

Source: Since FAIROS uses [F-UJI](https://f-uji.net) to assess a Research Object, this test is an instance of the test **FsF-F1-01MD-2**. You can find more information in the european project FAIR IMPACT(https://fair-impact.eu/metrics-data)
